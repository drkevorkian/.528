Seed corpus for `container_parser_demux_reader`.

These seed files are generated by `tools/fuzz_seed_gen` using `libsrs_mux`,
so they stay structurally valid as the container format evolves.

The generator writes files named **`seed_528_*`** (primary `.528` naming). If you have older **`seed_srsm_*`** files locally, regenerate or rename them after pulling a tree that includes the renamed seeds.

Regenerate:
`cargo run -p fuzz_seed_gen`

Run fuzz target with corpus:
`cargo fuzz run container_parser_demux_reader tests/fuzz/corpus/container_parser_demux_reader`
